I created this project to help farmers and rural people monitor their health easily. In many villages, hospitals are far away and regular health checkups are difficult. So I built a smart system that can check basic health conditions like temperature, pulse, and heart rate using IoT sensors.

The sensor data is sent to a Flask backend, where a machine learning model predicts whether the health condition is normal or needs attention. The results are shown in a React web application, and all patient data is stored in MongoDB for future reference.

This system can help farmers and rural communities detect health problems early and take action before the condition becomes serious. It can also be useful for remote monitoring in rural health centers.
